# SeaOfChaos 🎮

An underwater shooter game built in C++ where you control a submarine, fire torpedoes, and battle sea creatures.

## Features
- Submarine combat with torpedoes
- Multiple enemy types (Blue Whale, Coral enemies)
- Power-ups and scoring system

## How to Run
Open `SeaOfChaos.sln` in Visual Studio and build the solution.

## Controls
- Arrow Keys – Move submarine
- Space – Fire torpedo
